How to use image search

Place your query after the base URL Example: https://image2-patricksevat-1.c9users.io/YOUR QUERY
Upto 10 results are shown, for more results add ?offset={number} to your query. Example: https://image2-patricksevat-1.c9users.io/YOUR QUERY?offset=2
Or browse recent queries https://image2-patricksevat-1.c9users.io/recent

Results provided by Google Custom Search
